## Requirements  
* **[Raspberry Pi Zero 1 Codebase](./raspberry-pi-zero-1/)**

  * Requirements: Python 3.7.3

* **[Raspberry Pi Zero 2 Codebase](./raspberry-pi-zero-2/)**
  * Optimized for: Quad-core ARMv8 (M64-bit environment)
  * Requirements: Python 3.11+
